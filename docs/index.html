🚀 Quick Start

Smart Installation (Recommended)

# Clone the repository
git clone https://github.com/nadavsimba24/calclaw.git
cd calclaw

# Run smart installer with questionnaire
chmod +x scripts/install_smart.sh
./scripts/install_smart.sh

With Docker

# Build and run with Docker
./scripts/build_docker.sh
./scripts/run_with_docker.sh
