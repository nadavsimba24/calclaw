## 🚀 Quick Start Commands

### Clone and Install
```bash
# Clone the repository
git clone https://github.com/nadavsimba24/calclaw.git
cd calclaw

# Run smart installer (recommended)
chmod +x scripts/install_smart.sh
./scripts/install_smart.sh

# Or install personal edition
./scripts/install_personal.sh

# Or with Docker
./scripts/run_with_docker.sh
```

### Direct Links for Download
