# 🚀 Calclaw Smart Installer

## 📖 מבוא

**Calclaw Smart Installer** הוא מערכת התקנה חכמה שמתאימה את עצמה לצרכים שלך - **קלילה כמו OpenClaw, חזקה כמו מערכת ארגונית שלמה!**

המערכת שואלת אותך 5 שאלות פשוטות ומתקינה את **בדיוק מה שאתה צריך** - לא פחות, לא יותר.

## 🎯 למה Smart Installer?

### ✅ **התאמה אישית מלאה:**
- **לא צריך להתקין הכל** - רק מה שאתה באמת צריך
- **לא מבזבז משאבים** - מותאם למחשב שלך
- **לא מסובך** - שאלון פשוט של 5 שאלות

### ✅ **קלילה כמו OpenClaw:**
- **התקנה מהירה** - דקות בודדות
- **פשוטה לשימוש** - ממשק אינטואיטיבי
- **קלה לתחזוקה** - סקריפטים מובנים

### ✅ **חזקה כמו מערכת ארגונית:**
- **כל היכולות** - מאיי.איי מקומי עד אבטחה ארגונית
- **כל הסוגים** - משימוש אישי עד ארגון גדול
- **כל הפריסות** - ממחשב אישי עד Kubernetes

## 🚀 איך זה עובד?

### שלב 1: שאלון התאמה חכם
```bash
./install_calclaw_smart.sh
```

המערכת שואלת אותך 5 שאלות:

1. **מה המטרה העיקרית שלך?** (שימוש אישי, פיתוח, עסק, ארגון, רשות מקומית)
2. **איזה יכולות הכי חשובות לך?** (AI מקומי, ניהול משימות, Telegram, חיפוש תמונות, מערכת ארגונית, אבטחה)
3. **איפה תרצה להריץ את Calclaw?** (מחשב אישי, שרת מקומי, ענן, Kubernetes)
4. **מה רמת האבטחה שאתה צריך?** (מינימלית, בסיסית, מתקדמת, ארגונית)
5. **האם יש לך כבר מודלים של Ollama?** (צריך להתקין, יש לי, לא רוצה)

### שלב 2: ניתוח והתאמה
המערכת מנתחת את התשובות שלך ויוצרת **תוכנית התקנה מותאמת אישית**:

- **שימוש אישי:** קלילה, מהירה, פשוטה
- **פיתוח:** עם כלים, דוקומנטציה, דוגמאות
- **עסק קטן:** אוטומציה, אינטגרציה, עברית
- **ארגון גדול:** אבטחה, compliance, high availability
- **רשות מקומית:** חיפוש תמונות, ניתוח חזותי, ערים ישראליות

### שלב 3: התקנה חכמה
המערכת מתקינה **רק את מה שאתה צריך**:

- **Calclaw Core** - תמיד מותקן (הליבה)
- **Ollama** - רק אם אתה צריך AI מקומי
- **Task Automation** - רק אם אתה צריך ניהול משימות
- **Telegram Bot** - רק אם אתה צריך אינטגרציה
- **Vision Search** - רק אם אתה צריך חיפוש תמונות
- **Organizational AI** - רק אם אתה צריך מערכת ארגונית
- **Enterprise Security** - רק אם אתה צריך אבטחה מתקדמת

### שלב 4: הרצה וניהול
המערכת יוצרת **סקריפטים חכמים**:

- **`start_calclaw.sh`** - מפעיל את כל השירותים שלך
- **`stop_calclaw.sh`** - עוצר את כל השירותים שלך
- **`status_calclaw.sh`** - מציג סטטוס של כל השירותים שלך
- **`calclaw_config.json`** - קובץ תצורה מותאם אישית
- **`README_CUSTOM.md`** - דוקומנטציה מותאמת אישית
